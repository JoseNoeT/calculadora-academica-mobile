# Calculadora Académica Mobile

Aplicación móvil offline-first para calcular, organizar y proyectar notas académicas.

## Objetivo

Ayudar a estudiantes a conocer su avance académico, calcular notas necesarias para aprobar y simular escenarios futuros desde el celular.

## Stack planificado

- React Native
- Expo
- TypeScript
- SQLite
- AsyncStorage / MMKV
- Jest

## Estado del proyecto

Proyecto en fase de configuración inicial, arquitectura y diseño de experiencia.
